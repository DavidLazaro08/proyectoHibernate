# proyectoHibernate

Este proyecto se ha construido tomando como referencia los ejercicios de 
Hibernate vistos en clase. Los ejemplos originales incluían únicamente la 
configuración básica y algunas entidades simples, por lo que se ha ampliado 
el diseño para incorporar una arquitectura por paquetes (DAO, entities, util) 
y un sistema completo de CRUD con relaciones OneToMany, ManyToOne y ManyToMany.
